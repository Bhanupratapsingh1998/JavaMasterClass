Q6. Write an SQL query to fetch the 3rd highest salary from the employee table.
SELECT DISTINCT salary
FROM employee
ORDER BY salary DESC
LIMIT 1 OFFSET 2;

Q7. Write an SQL query to fetch employees whose salary is greater than their manager’s salary.
SELECT *
FROM employee emp
INNER JOIN employee manager
    ON emp.managerid = manager.id
WHERE emp.salary > manager.salary;
